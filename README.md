## 说明
简单Conda工具，提供不同平台conda的安装，加载，备份等功能。


## 安装

```
pip install xy_conda
```

## 使用

###### 1. 命令行

```
> xy_conda -w backup
> 是否备份当前环境 python_3_11_3 
> 输入 [Y/n]
> Y

> xy_conda -w backup -n conda_name
> 是否备份当前环境 python_3_11_3 
> 输入 [Y/n]
> Y
```
